### Installation

1. #### (_Optional_) Create a Python virtual environment for the solutions
   
   ```bash
   python3 -m venv <your_venv_name>
   ```

2. #### Install the required dependencies from the (_requirements.txt_) file
   
   ```bash
   pip3 -r install requirements.txt
   ```
